# Candidature BTS 2025

Ce repository contient les éléments relatifs à ma candidature pour plusieurs BTS dans le domaine de l'informatique et de l'électronique. Vous y trouverez des lettres de motivation, des CV et des projets personnels que j'ai développés dans le cadre de mes études et formations.

## Objectif

L'objectif principal de ce repository est de centraliser tous les documents et projets liés à ma candidature pour le BTS **Services Informatiques aux Organisations** (SIO) en apprentissage, ainsi que d'autres formations en **cybersécurité, informatique et réseaux**.

## Structure du Repository

- `lettre_de_motivation/`: Contient les lettres de motivation pour différents BTS.
  - `BTS_SIO_Lycée_Marie_Curie.md`: Lettre de motivation pour le BTS SIO au Lycée Marie Curie.
  - `BTS_Cybersecurite_Lycée_Rempart.md`: Lettre de motivation pour le BTS Cybersécurité au Lycée du Rempart.
  
- `cv/`: Mon CV au format PDF et autres formats.
  - `CV_MissterFreeze24.pdf`: Mon CV à jour.

- `projets/`: Projets personnels réalisés dans le cadre de ma formation.
  - `Projet_Cybersecurite/`: Projet sur la cybersécurité.
  - `Projet_Developpement_Web/`: Projet en développement web full stack.
Vous trouverez dans ce dépot un fichier qui répertorie tous les MOOCs auquel je suis inscrit et documentations que j'ai lu.

## Prérequis

- **Git** pour cloner ou télécharger le repository.
- **Markdown** compatible viewer pour afficher les fichiers `.md` correctement.
- Un environnement de développement adéquat pour tester ou modifier les projets (par exemple, Node.js pour les projets web).
